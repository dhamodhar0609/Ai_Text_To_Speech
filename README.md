# AI Text-to-Speech Converter

A web-based AI Text-to-Speech application built using HTML, CSS, and JavaScript.  
It converts user-entered text into speech using the Web Speech API with dynamic voice selection.

## Features
- Real-time text to speech conversion
- Multiple voice selection
- Responsive UI design
- Client-side (No backend required)

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Web Speech API
